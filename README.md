
# 📸 React Gallery

A simple and responsive image gallery built with **React** that fetches images from the **Picsum Photos API** and displays them in a clean layout.

## ✨ Features

* Fetches images dynamically from the **Picsum Photos API**
* Responsive gallery layout
* Lazy loading for improved performance
* Clean and modern UI using **Tailwind CSS**

## 🛠️ Technologies Used

* ⚛️ React
* 🎨 Tailwind CSS
* 🌐 Axios (for API requests)
* 🖼️ Picsum Photos API

## 🎯 Purpose

This project was built to practice:

* Fetching data from a REST API
* Working with React state and rendering lists
* Handling asynchronous requests
* Building responsive user interfaces with Tailwind CSS

## 🔗 API

Images are fetched from **[Picsum Photos](https://picsum.photos)**.

## 🚀 Getting Started

```bash
npm install
npm run dev
```
